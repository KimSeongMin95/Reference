[===== 구분 =====]
    - SBCS(Single-Byte Character Set)
        단일 문자 집합으로 문자를 표현하는 데, 1바이트를 사용합니다.

    - MBCS(Multi-Byte Character Set)
        복수 문자 집합으로 유니코드를 제외한 문자 인코딩에서 두 개 이상의 바이트를 사용해서 문자 를 표시하는 방식입니다.

    - WBCS(Wide-Byte Character Set) 
        확장형 문자 집합으로 유니코드입니다.


[===== 기본 =====]
    - 표준 BCD 코드(Binary Coded Decimal)
        6비트

    - EBCDIC 코드(Extended Binary Coded Decimal Interchange Code)
        8비트

    - ASCII 코드(American Standard Code for Information Interchange)
        7비트

    - ANSI 코드(American National Standard Institute/Extended ASCII)
        8비트

        2바이트, 11,172자의 한글을 표현
